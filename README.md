# College ERP - Frontend

A modern, responsive College Management System built with React, Material UI, and Vite.

## Features
- **Admin**: Dashboard, Student List, Fee Management
- **Teacher**: Dashboard, Attendance Marking, Leave Application
- **Student**: Dashboard, Fee Status, Timetable

## Tech Stack
- React 18
- Material UI (v5)
- Zustand (State Management)
- Recharts (Data Visualization)
- Vite PWA

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173`
